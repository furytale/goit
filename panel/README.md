## 1. Create Panel component
 - Use the next html structure:
```
 <div class="panel panel-primary">
   <div class="panel-heading">
     <h3 class="panel-title">Panel primary</h3>
   </div>
   <div class="panel-body">
     Panel content
   </div>
 </div>
```
